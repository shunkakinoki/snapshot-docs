### Get a single space

#### Arguments

id `string`

#### Example

{% tabs %}
{% tab title="Request" %}
```graphql
query {
  space(id: "yam.eth") {
    id
    name
    about
    network
    symbol
    members
  }
}
```

### Get multiple spaces

#### Arguments

first `number`  
skip `number`  
orderBy `string`  
orderDirection `asc` or `desc`

#### Example

{% tabs %}
{% tab title="Request" %}
```graphql
query {
  spaces(
    first: 20,
    skip: 0,
    orderBy: "created",
    orderDirection: asc
  ) {
    id
    name
    about
    network
    symbol
    strategies {
      name
      params
    }
    admins
    members
    filters {
      minScore
      onlyMembers
    }
    plugins
  }
}
```
{% endtab %}
